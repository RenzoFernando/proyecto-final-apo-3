# Dataset original de Kaggle

Las imágenes utilizadas como base para el proyecto fueron obtenidas del dataset **Fruit Quality Classification** disponible en Kaggle:

https://www.kaggle.com/datasets/ryandpark/fruit-quality-classification

Para reproducir el conjunto de datos utilizado en este proyecto, se debe descargar el dataset específicamente la carpeta:

`mixed_quality`

Dado que las imágenes de esta carpeta contienen múltiples frutas por fotografía, fue necesario realizar un proceso de segmentación y recorte para obtener imágenes individuales de cada fruto. Una vez completado este proceso, las imágenes pueden utilizarse en el flujo de preparación y entrenamiento descrito en el proyecto.
